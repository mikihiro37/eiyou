## 1. 症状検索の仕組み

ユーザーが症状ワードを入力すると、各栄養素の `deficiency`（欠乏症状）・`excess`（過剰症状）フィールドに対して部分一致検索を実施。  
ヒットした栄養素をグラフ上でハイライトし、「少ない状態との関連 / 多い状態との関連」として表示する。

**実装上の安全措置（確認済み）**

- 検索結果バナーに「症状の原因を判定するものではありません」と明示（`index.html` L1319）
- 詳細パネルに「診断・治療・疾病予防を目的とするものではありません。個別の判断は専門職にご相談ください。」（L754）
- ハイライト色は「低(青)=少ない状態との関連 / 高(橙)=多い状態との関連 / ?(灰)=要確認」として色+テキスト双方で表示
