## 補足：ローディングtips の表示文脈

- 表示タイミング: Gemini API 解析中のオーバーレイ（`showLoading` 関数 → `NUTRITION_TIPS` ローテーション）
- ユーザーコンテキスト: 解析処理の待機中に表示される豆知識として機能
- 個別ユーザーへの判定・診断との混同リスク: 低（解析結果と分離された待機画面での表示）
- ただし「〜の原因」「〜不足は〜」の表現は、解析結果と誤解されるリスクがゼロではない
