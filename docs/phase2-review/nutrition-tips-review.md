## 5. 関連デッドコード：getBarLabel 関数

`checker/data.js` L374〜380 に `getBarLabel()` 関数が存在するが、現在呼び出し箇所がない。

```javascript
// バーチャートのラベル（デッドコード — 今後の表示拡張用に保持）
// TODO Phase 1b-3: 使用箇所を確認し、削除または統合する
function getBarLabel(pct) {
  if (pct < 50) return '不足';
  if (pct < 80) return 'やや不足';
  if (pct <= 120) return '適量';
  if (pct <= 150) return 'やや過剰';
  return '過剰';
}
```

返り値の「不足」「やや不足」「やや過剰」「過剰」は、現行の表示方針（参考基準より少なめ/多め等）とは整合しない可能性がある。

**フラグ**: 将来的にこの関数を有効化する場合は、返り値の表現をアプリの表示方針に合わせて見直す必要がある。
